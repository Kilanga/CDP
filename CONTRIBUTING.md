# Contributing

Merci de contribuer à `CDP` ! Quelques règles simples :

- Ouvre d'abord une issue pour discuter des changements importants.
- Pour proposer des modifications, fais une branche depuis `main` nommée `feature/xxx` ou `fix/yyy`.
- Fais une PR ciblant `main` avec une description claire et des étapes pour reproduire.
- Respecte le style existant du projet et ajoute des tests si possible.

Tests et vérification locale

1. Installe les dépendances : `bundle install`.
2. Lance la suite de tests (s'il y en a) : `bundle exec rake test`.
